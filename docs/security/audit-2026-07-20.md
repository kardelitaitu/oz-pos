### Mitigations

- **unic-*** crates: Unicode character property databases. Used indirectly through `idna` / `url`. No known exploits — unmaintained status means no security patches for future CVEs. Acceptable risk for now.
- **glib**: The unsound `Iterator` impl for `VariantStrIter` can cause UB. Only triggered when iterating `Variant` children — not used in OZ-POS code paths. Tauri 2.x will ship updated glib.
- **spin**: Yanked versions are not pulled by `cargo update`. Pinning via `[patch.crates-io]` in workspace `Cargo.toml` ensures 0.9.9+ is used.

### CI Gate

Added `cargo-audit` CI job (non-blocking) that runs on every PR. Known issues are documented here with rationale.
