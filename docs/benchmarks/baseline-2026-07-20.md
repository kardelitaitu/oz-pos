# OZ-POS Performance Baselines — 2026-07-20

> Generated via `cargo bench -p oz-core` (Criterion.rs, 100 samples per benchmark).
> All times in **microseconds** (µs) unless otherwise noted. Lower is better.

## Hardware

- CI runner: GitHub Actions `ubuntu-latest` (4 vCPU, 16 GB RAM)
- Expected range accounts for CI variability (±15%)

## How to use

```bash
# Run all benchmarks
cargo bench -p oz-core

# Open HTML report
open target/criterion/report/index.html

# Compare against stored baseline
cargo install critcmp
critcmp baseline current
```
