# Performance Benchmarks — 0.0.14

## Criterion Benchmarks

Run with:
```bash
cargo bench -p oz-core
```

### Money Arithmetic (money_bench)

| Benchmark | Expected | Notes |
|-----------|----------|-------|
| `money_checked_add` | < 10ns | i64 add + currency compare |
| `money_checked_sub` | < 10ns | i64 sub + currency compare |
| `money_checked_mul` | < 10ns | i64 mul + overflow check |
| `money_checked_div` | < 15ns | i64 div + overflow/zero check |
| `money_serde_roundtrip` | < 200ns | JSON parse + serialize |

### Cart Operations (cart_bench)

| Benchmark | Expected | Notes |
|-----------|----------|-------|
| `cart_add_line` | < 100ns | Push to Vec, currency check |
| `cart_total_20_items` | < 1µs | 20x checked_add + discount |
